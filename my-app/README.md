## ⚙️ Getting Started

### Clone the repository

```bash
git clone https://github.com/parthnai21/My-Portfolio
```

### Navigate to the project

```bash
cd My-portfolio
```

### Install dependencies

```bash
npm install
```

### Run the development server

```bash
npm run dev
```

### Create a production build

```bash
npm run build
```
